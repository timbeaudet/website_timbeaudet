## What was learned {.wp-block-heading}

### Remote Triggering of Build System is Lacking {.wp-block-heading} In an attempt to continue working while traveling to family I started moving Microsoft headers into a platform includes header for TurtleBrains, while working on a MacBook Pro. With no way to run the build except nightly emails it took a bit to ensure the build was not broken. Having the ability to trigger the build system 

**and** run deploys from remote locations would be extremely useful. 

### Streaming from Rural Maine {.wp-block-heading} It is possible to stream from Maine in a rural environment, though the quality level does drop. Still 800 kbps upload for programming seemed to work okay.
